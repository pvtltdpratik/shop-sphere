# ShopSphere - E-Commerce Full-Stack Application

A complete, production-ready e-commerce platform built with React, Firebase, and Razorpay.

## Features

✅ **User Features**
- Google authentication
- Browse products with advanced filters and search
- Product details with image gallery
- Shopping cart with persistent storage
- Checkout with Razorpay payment integration
- Order tracking with real-time status updates
- User dashboard with order history

✅ **Admin Features**
- Complete product management (CRUD)
- Order management with status updates
- User management dashboard
- Sales analytics and statistics
- Real-time order updates

✅ **Technical Features**
- Firebase Firestore database
- Firebase Storage for images
- Real-time data synchronization
- Responsive design (mobile-first)
- Dark/Light mode ready
- Toast notifications
- Skeleton loaders
- Error handling

## Tech Stack

- **Frontend**: React 18, React Router v6, Tailwind CSS
- **State Management**: React Context API + useReducer
- **Backend**: Firebase (Firestore, Auth, Storage)
- **Payment**: Razorpay
- **UI Components**: Lucide React Icons
- **Build Tool**: Vite

## Prerequisites

- Node.js (v16+)
- npm or yarn
- Firebase account
- Razorpay account (test mode available)
- Google Developer account
